Systembeschreibung - SvxLink / SvxReflector

SvxReflector (früher auch "östkustlänken" genannt) ist ein nationales Netzwerk schwedischer analoger Repeater, das mit Hilfe von SvxLink in der Regel über das Internet miteinander verbunden ist.

Im Netzwerk Speech werden Gruppen verwendet , um den Datenverkehr zu trennen. Eine Sprachgruppe kann mit einem logischen "Kanal" verglichen werden.

  • Ein Repeater kann mehrere Sprachgruppen aufnehmen (überwachen). Welche Sprachgruppen für jeden Repeater geeignet sind, finden Sie auf der Homepage von SvxPortalen.
  • Eine Sprachgruppe kann vom lokalen Benutzer mithilfe von DTMF-Befehlen oder Subton (CTCSS) manuell ausgewählt werden.
  • Der Repeater kann auch eine lokale Standardnummerngruppe haben, die beim lokalen Start automatisch aktiviert wird (wenn der Benutzer keine aktive Auswahl trifft).

Auf der anderen Seite kann ein Repeater nie seine aktiv auf mehr als eine Sprachgruppe zu einem Zeitpunkt.

Eingehender Datenverkehr in einer überwachten Sprachgruppe startet den Repeater automatisch, kann jedoch normalerweise während eines laufenden QSO nicht unterbrochen werden.

Repeater und Simplex-Knoten im Netzwerk überwachen statisch:

  • Sprachgruppe 240, die ganz Schweden abdeckt.
  • Eigene Distriktnummerngruppe - zB 2405 für SM5.
  • Mögliche Repeater und Distrikte in der Nähe.
  • Möglicherweise eigene lokale Nummerngruppe - zB 240515 für Eskilstuna. Dies kann dann verwendet werden, um nur diesen Repeater zu "adressieren".
  • Möglicherweise Clusternummerngruppe - zB 24078 für SK7RFL + SK7RN an Land oder 24021 für die "Kalix-Linie".

Das gesamte System kann in Echtzeit über das Svx-Portal (diese Website) überwacht werden.

Die neuesten Informationen finden Sie in einer Mailingliste: https://svxreflector-sm.groups.io/

Wahl der Sprachgruppe

Wenn Sie eine bestimmte Sprachgruppe verwenden möchten, können Sie diese auf zwei Arten auswählen - mit DTMF oder Subton (CTCSS).

Wenn in der ausgewählten Sprachgruppe kein Datenverkehr vorhanden ist, wechselt das System nach 30 Sekunden in den Leerlaufmodus (einstellbar über Sysop) und wartet auf die Aktivität in überwachten Sprachgruppen. Ein kleines "FLIRRP" ist zu hören, wenn der Repeater oder Simplex-Knoten in seine Standardposition zurückkehrt. Bei SK7RFL, SK7RN und SK5BN geschieht dies immer in Verbindung mit dem Herunterfahren des Repeaters.

DTMF

Um die Sprachgruppe 2405 mit DTMF zu aktivieren, drücken Sie: 

912405#

Der Repeater gibt bekannt, dass er zur Sprachgruppe 2405 wechselt, wenn Sie PTT loslassen.

Subton (CTCSS)

Eine Liste der Sprachgruppen, die mit Subton aktiviert werden können, finden Sie hier im SvxPortal .

Um beispielsweise die Sprachgruppe 2405 mit Subton, CTCSS zu aktivieren, wählen Sie Subton 91,5 Hz. Nach der ersten Sendesitzung gibt Tepeatern bekannt, zu welcher Sprachgruppe gewechselt wird, wenn Sie PTT freigeben.

Die Aktivierung mit einem Unterton kann nur im Schlafmodus des Repeaters erfolgen, dh nicht während des aktuellen QSO.

Trinkgeld

Programmieren Sie mehrere Speicherkanäle mit unterschiedlichen Untertönen, um den Repeater / Simplex-Knoten mit der richtigen Sprachgruppe direkt aktivieren zu können.

Für Knoten mit einem Subton, der mit keiner Sprachgruppe verbunden ist, wird in der Spalte "Lokal" die Spalte "Lokal" angezeigt. In der Liste können Dienste auf dem Knoten verwendet werden, ohne dass eine Sprachgruppe verbunden ist. Es kann praktisch sein, verschiedene Module in SvxLink wie Papegoja, EchoLink, Voicemail usw. zu verwenden. Es ermöglicht auch, lokale QSOs auf dem Kanal zu haben, ohne dass eine Sprachgruppe von außen aktiviert wird.

Ähnlich wie bei einer aktiven Sprachgruppe muss der Kanal eine Zeit lang ruhen, bevor die Verbindung von Sprachgruppen sowohl remote als auch mit einem Unterton hergestellt werden kann. Die Zeit ist per Sysop einstellbar, beträgt aber standardmäßig 30 Sekunden.

Es ist jedoch jederzeit möglich, die Sprachgruppe mit DTMF-Befehlen zu aktivieren / ändern.

QSY

Während eines QSO für eine Sprachgruppe können alle aktiven Knoten in eine andere Sprachgruppe verschoben werden. Der aktive Knoten ist der Knoten, der seit Aktivierung der Sprachgruppe lokal aktiv war, dh die Lärmschutzwand hat sich mindestens einmal geöffnet.

Die QSY-Funktion wird durch DTMF-Befehle gesteuert.

92 # QSY zu einer Zufallszahlengruppe, die vom Reflektorserver automatisch ausgewählt wird
92nnn # QSY zur Sprachgruppe nnn
93 # Folgen Sie dem neuesten QSY

QSY für temporäre Sprachgruppen wird in geeigneter Weise so verwendet, dass Anrufe zu einem Anfang in einer breiteren Sprachgruppe getätigt werden, z. B. 240 für ganz Schweden. Wenn alle Teilnehmer "angerufen" haben In der Sprachgruppe wird der QSY-Befehl ( 92 # ) verwendet, um alle aktiven Knoten in die temporäre Sprachgruppe zu verschieben. Die breitere Nummerngruppe ist dann wieder frei für Anrufe.

Nach 5 Minuten wird auch eine automatische QSY für die Sprachgruppe 240 durchgeführt.

QSY zu einer bestimmten Sprachgruppe kann verwendet werden, um einem laufenden QSO weitere Knoten hinzuzufügen. Angenommen, ein QSO wurde in Talgroup 2403, dh SM3, gestartet. Unter dem QSO möchten Sie auch mit Knoten in SM5 erweitern, dh mit der Nummerngruppe 2405. Hierfür kann dann der Befehl 922405 # verwendet werden. Alle Knoten, die derzeit auf 2403 aktiv sind, werden auf 2405 verschoben, und alle Knoten, die 2405 überwachen, stellen eine Verbindung zur Sprachgruppe her.

In neueren Versionen von SvxLink können Sie einem QSY sehr einfach folgen. Wenn QSY auftritt, sagt der Repeater " QSY 24099nn ausstehend ". Wenn Sie PTT innerhalb von ca. 15 Sekunden drücken, folgt der Repeater diesem QSY. Andernfalls sagt der Repeater " QSY [24099nn] ignoriert ".

Ein passiver Knoten kann auch manuell erstellt werden, um dem neuesten QSY mit dem Befehl 93 # zu folgen . Dies kann hilfreich sein, wenn Sie vor QSY keine Zeit zum Senden hatten oder innerhalb von 15 Sekunden nach QSY Zeit hatten, PTT zu drücken.

Beachten Sie den Unterschied zwischen QSY und lokaler Wahl von TG.

  • 922405# verschiebt alle aktiven Knoten in die Sprachgruppe 2405 (QSY).
  • 912405# verschiebt nur den lokalen Knoten zur Sprachgruppe 2405 ( Sprachgruppenauswahl ).

Verletzung

Die Zuordnung zwischen Untertönen und Sprachgruppen entspricht der SSA-Liste für CTCSS. Der zweite Ton in jedem Distrikt ("Sekundär 1") wird der Sprachgruppe für diesen Distrikt zugeordnet.

Wenn Sie eine Sprachgruppe direkt trennen möchten, ohne 30 Sekunden warten zu müssen, können Sie 910 # (TG0) drücken. Gut zu verwenden, wenn Sie zufällig die falsche Sprachgruppe auswählen.

Sie können nicht mehrere Sprachgruppen gleichzeitig aktivieren.

Wenn Sie eine bestimmte Sprachgruppe aktiviert haben, z. B. TG2403, wird der Datenverkehr von anderen Sprachgruppen blockiert, bis der Datenverkehr auf TG2403 endgültig gestoppt wurde und 30 Sekunden vergangen sind.

SvxReflectorn ist nicht mit DMR / Brandmeister verbunden, verwendet jedoch ähnliche Namen und Nummern für Talgrupper.
